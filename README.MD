## GUI for Pytest based on appJar.
#### Clone project:
```
git clone https://github.com/vsshk/pytest-gui.git
cd pytest-gui
```
### install requirements and Allure
```
$ pip install -r requirements.txt 
```
* [install Allure](https://docs.qameta.io/allure/)
* ![](demo-gif.gif)
# How to run?
```
$ python start.py
```
